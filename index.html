Cyrill Stachniss SLAM Course Solutions

  • MATLAB solution code for the homework assignment of cyrill stachniss SLAM course.
  • Implementation includes EKF SLAM, UKF SLAM, Gridmaps, Particle Filter, Odometry Calibration, Fast-SLAM, Graph-Based SLAM.

    Simultaneous Moving Object Moving Transmitter Localization with Unknown Offsets

  • Investigated the multistatic localization of a moving object in position and velocity using an uncoordinated moving transmitter.
  • The measurements are time delays and frequencies and each kind is subject to an unknown amount of offset.
  • Derived the condition about the localization geometry that can eliminate the degradation due to the offsets.
  • Developed algebraic solution to simultaneously localize the moving object and transmitter that can reach the CRLB accuracy.
  • Examined the particular case of having time delay measurements only and devised the optimal geometry.

    • Simultaneous Target and Transmitter Localization

  • Investigated the object localization in the absence of the transmitter position.
  • Examined the performance improvement compared to the TDOA technique.
  • Developed algebraic closed-form solution for the nonlinear problem of simulataneous target and transmitter localization.
  • Derived the optimum receiver placement in the absence of transmitter position.

    Elliptic or Hyperbolic Localization Using Minimum Measurement Solutions

  • Derived a new algebraic positioning solution using minimum number of measurements.
  • Reduced the computation to the roots of a quadratic equation.
  • Developed an outlier detector based on graph processing.
  • Proposed a best linear unbiased estimator (BLUE) to form the final location estimate that can reach the CRLB accuracy.
  • Developed a measurement partitioning scheme to improve the performance when the noise level becomes large.
